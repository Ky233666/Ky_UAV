# 用户指南

## 1. 运行环境

- Unity：`2022.3.62f3`
- 默认主场景：`Assets/Scenes/Main/MainScene.unity`
- 自定义障碍物实验场景：`Assets/Scenes/Sandbox/CustomObstacleSandbox.unity`

## 2. 首次打开项目

1. 使用 Unity Hub 打开项目。
2. 执行 `Tools/KY UAV/Bootstrap Delivery Assets`。
3. 打开 `Assets/Scenes/Main/MainScene.unity`，或执行 `Tools/KY UAV/Create Or Refresh Custom Obstacle Sandbox Scene` 后打开 `Assets/Scenes/Sandbox/CustomObstacleSandbox.unity`。
4. 进入 Play Mode。

## 3. 任务点操作

任务点交互依赖主场景原有 Canvas 按钮，而不是右侧运行时控制面板。

- `新增任务点`
  - 点击后进入放置模式。
  - 在地面空旷区域点击左键放置。
  - `Esc` 取消。
- `导入任务`
  - 从 `Assets/Resources/taskpoints.csv` 导入默认任务。
- `清空任务`
  - 清除当前场景任务点。

## 4. 起飞点操作

起飞点交互位于右侧运行时控制面板的“起飞点”区域。

- `新增`
  - 点击后进入放置模式。
- `移动`
  - 先选中已有起飞点，再点新位置。
- `删除`
  - 点击已有起飞点删除。
- `清空`
  - 删除全部手动起飞点。

## 5. 自定义障碍物操作

障碍物交互位于右侧运行时控制面板的“障碍物”区域，且仅允许在 `Idle` 状态编辑。

- `绘制`
  - 点击后进入绘制模式。
  - 在空白地面按住左键拖拽生成建筑。
  - 若拖拽区域与现有建筑、任务点或起飞点重叠，系统会阻止创建。
  - `Esc` 取消。
- `样式`
  - 在 `长方体 / 城市楼体模板` 之间切换。
  - 楼体模板来自 `SimpleCityPackage` 现成建筑资源。
  - 新建楼体会自动贴地，并在创建后立即纳入障碍缓存与路径规划检查。
- `缩放`
  - 调整下一次创建建筑的统一缩放倍率。
  - 会同时影响绘制预览、碰撞校验和最终生成尺寸。
- `删除`
  - 点击后进入删除模式。
  - 点击已有自定义建筑即可删除。
  - 固定场景建筑不支持删除。
- `清空`
  - 删除全部自定义建筑。
- `高度`
  - 调整下一次创建建筑的默认高度。

## 6. 仿真控制

- `开始仿真`
  - 分配任务并启动机群。
- `暂停`
  - 暂停当前仿真。
- `重置`
  - 重置无人机、任务状态、路径和统计。
- `重建机群`
  - 按当前无人机数量重新生成机群。

快捷键：

- `F5`：开始/继续
- `F6`：暂停
- `F7`：重置
- `F8`：重建
- `F10`：退出

## 7. 算法与规划参数

右侧运行时控制面板支持：

- 调度算法切换
- 路径规划算法切换
- 路径规划算法过程演示
- 无人机数量调整
- 飞行速度调整
- 仿真倍速调整
- 规划网格大小调整
- 规划边界调整
- 对角搜索开关
- 障碍自动配置开关
- 规划线与航迹显示开关

## 8. 算法过程演示

右侧运行时控制面板新增“算法过程演示”区域，用于回放当前路径规划算法的搜索过程。

当前已接入过程演示的规划算法：

- `StraightLine`
- `AStar`
- `RRT`

演示模式：

- `仅最终结果`
- `完整过程`
- `关键步骤`

可执行操作：

- `播放/继续`
- `暂停`
- `单步`
- `重置`
- 切换当前无人机
- 切换演示模式
- 切换播放速度 `0.5x / 1x / 2x / 5x`

界面说明：

- 状态卡显示当前无人机、算法名称、步骤数和演示状态
- 描述卡显示当前步骤解释，便于说明“这一帧算法正在做什么”
- 图例卡说明颜色和线条含义，重点区分起点、终点、当前扩展节点、已访问节点、候选路径、回溯路径和最终路径

快捷键：

- `F9`：播放/暂停算法过程演示
- `F11`：算法过程单步
- `F12`：重置算法过程演示

## 9. 相机与 2D 轨迹检查

相机模式：

- `1`：总览
- `2`：跟随
- `3`：`2D俯视`
- `Tab / E`：下一架无人机
- `Q`：上一架无人机

`2D俯视` 用于直接检查 `XZ` 平面路径：

- 轨迹线和规划线会投影到统一平面
- 右侧摘要会显示建筑告警数
- 若轨迹穿越建筑 footprint，对应轨迹会切换为告警样式

交互说明：

- 当鼠标悬停在右侧运行时控制面板、Scroll View、输入框或下拉控件上滚动滚轮时，场景相机不会响应缩放。
- 鼠标移回非 UI 区域后，场景滚轮和视角控制恢复正常。

## 10. 导出与实验

结果导出位于运行时控制面板“结果”区域。

- `导出CSV`
- `导出JSON`
- `新会话`
- `应用目录`
- `默认目录`

快捷键：

- `Ctrl+Shift+N`：新建导出会话
- `Ctrl+Shift+C`：导出 CSV
- `Ctrl+Shift+J`：导出 JSON
- `Ctrl+Shift+B`：开始批量实验
- `Ctrl+Shift+X`：停止批量实验

## 11. 运行时实验中心

实验中心支持四类预设：

- `Scheduling`
- `Planning`
- `Scaling`
- `Density`

可执行操作：

- 切换实验分组
- 切换具体预设
- 将预设应用到当前配置
- 直接启动预设批量实验
- 按当前运行时配置启动批量实验

## 12. 已知限制

- 任务点管理和起飞点管理分属两套交互入口。
- 自定义障碍物支持长方体和楼体模板，但当前不提供运行时旋转编辑。
- 自定义障碍物为运行时会话级对象，退出 Play Mode 或程序重启后不会自动写回场景文件。
- 批量实验结果主要通过文件查看；当前过程回放面向单次规划轨迹，不提供批量实验结果总览回放界面。
- 多机避让仍是第一版局部规则，复杂密集场景下可能出现等待抖动。
- 当前仍不支持运行时多场景切换。
- 算法过程演示当前按单无人机查看，不支持双算法并排同步回放。
