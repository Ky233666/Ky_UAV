# 答辩演示脚本

## 演示目标
- 在 `3` 分钟内展示“任务配置、算法切换、运行观察、结果导出”闭环。

## 建议流程
1. 打开主场景并进入运行。
2. 简要说明右侧运行面板已经替代大部分 Inspector 调参。
3. 展示当前任务点与起飞点，可补充新增一个任务点。
4. 先用 `PriorityGreedy + AStar` 开始仿真。
5. 切换到跟随镜头，再切下一架无人机，说明路径与轨迹显示。
6. 停止并切换另一组算法，说明调度或路径差异。
7. 展示统计区块里的任务进度、飞行距离、等待次数、冲突次数。
8. 点击导出 CSV / JSON，说明系统支持论文实验留档。
9. 说明批量实验可按预设重复运行，并生成 `session_manifest.json` 与 `session_summary.csv`。

## 话术重点
- 当前版本目标是“稳定交付”，不是继续扩张高风险功能。
- 系统已经形成多无人机调度、路径规划、运行时交互、实验导出闭环。
- 算法对比与批量实验数据可直接支撑论文结果整理。
