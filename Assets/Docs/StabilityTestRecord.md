# 稳定性测试记录

## 目标
- 连续 `10` 次 `开始 -> 完成/重置`
- 连续 `5` 次 `开始 -> 暂停 -> 继续`
- 连续 `3` 次 `重建机群 -> 重新开始`

## 记录模板

| 日期 | 用例 | 轮次 | 结果 | 阻塞性错误 | 备注 |
| --- | --- | --- | --- | --- | --- |
| 待填 | 开始-完成/重置 | 1/10 | 待测 | 无 |  |
| 待填 | 开始-暂停-继续 | 1/5 | 待测 | 无 |  |
| 待填 | 重建机群-重新开始 | 1/3 | 待测 | 无 |  |

## 关注点
- 无人机不会卡死在 `Waiting`。
- 批量实验中途不会丢失导出会话目录。
- 重置后轨迹、路径、等待次数、冲突次数和计时均清零。
- 重建机群后相机和运行面板仍可继续工作。
