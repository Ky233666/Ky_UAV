# 毕业设计项目进展报告

**项目名称**：基于 Unity 的无人机路径规划与调度算法仿真系统设计与实现
**完成日期**：2026年3月16日
**开发环境**：Unity 2022.3.x + C#

---

## 一、项目背景与任务目标

本项目旨在构建一个**轻量化、模块化的多无人机协同巡检仿真验证平台**，支持：
- 多无人机、多任务点的路径规划与调度
- 核心算法的运行时切换（A*、RRT、贪心、遗传等）
- 性能指标（能耗、耗时、冲突次数）的实时可视化
- 算法对比实验与数据导出

当前处于**第三阶段（3月）**——核心算法实现阶段，但整体进度**滞后于原计划**。

## 五、当前系统功能现状

### 可正常运行的功能
1. **多无人机生成**：DroneManager 自动生成指定数量无人机
2. **任务点管理**：创建、清除、导入任务点
3. **基础飞行控制**：无人机按队列依次飞向任务点
4. **仿真控制**：开始、暂停、重置
5. **视角切换**：总览/跟随相机

### 尚不能实现的目标
1. ❌ **路径规划算法**：无法演示 A*、RRT 等避障路径
2. ❌ **调度算法对比**：无法对比贪心、遗传等算法性能
3. ❌ **性能指标评估**：无法量化能耗、耗时、冲突次数
4. ❌ **数据导出**：无法生成对比实验数据表格

---

## 六、进度滞后原因总结

1. **时间管理不当**：第一、二阶段耗时超过预期，导致第三阶段时间压缩
2. **架构设计滞后**：先实现功能再补架构，导致后期重构成本高
3. **算法实现困难**：路径规划与调度算法需要较长的开发与调试周期
4. **测试不充分**：缺少系统级的集成测试与对比实验验证

## 八、结论

当前项目进度**滞后于原计划**，第三阶段（核心算法实现）尚未正式启动。主要原因是：
