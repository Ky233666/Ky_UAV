# 功能测试表

| 编号 | 功能项 | 步骤 | 预期结果 | 结果 |
| --- | --- | --- | --- | --- |
| F-01 | 主场景启动 | 打开 `MainScene` 进入 Play Mode | 运行面板自动创建，系统状态为就绪 | 待测 |
| F-02 | 开始仿真 | 点击开始仿真 | 自动导入或读取任务，机群进入运行中 | 待测 |
| F-03 | 暂停/继续 | 点击暂停，再点击开始 | 状态在已暂停/运行中间正确切换 | 待测 |
| F-04 | 重置仿真 | 点击重置 | 无人机、任务、轨迹、统计恢复初值 | 待测 |
| F-05 | 重建机群 | 修改数量并点击重建 | 机群按数量重生，仍可继续仿真 | 待测 |
| F-06 | 调度切换 | 切换调度算法后开始仿真 | 分配结果与运行摘要发生可观察变化 | 待测 |
| F-07 | 规划切换 | 切换路径规划后开始仿真 | 路径形态与轨迹显示发生变化 | 待测 |
| F-08 | 任务点新增 | 进入新增任务点模式并点击地面 | 成功生成任务点且位置合法 | 待测 |
| F-09 | 任务点导入 | 点击导入任务 | `taskpoints.csv` 中任务被正确导入 | 待测 |
| F-10 | 起飞点编辑 | 新增/移动/删除起飞点 | 起飞点编号、排序和位置更新正确 | 待测 |
| F-11 | 导出 CSV | 点击导出 CSV | 当前会话目录生成实验记录 CSV | 待测 |
| F-12 | 导出 JSON | 点击导出 JSON | 当前会话目录生成实验详情 JSON | 待测 |
| F-13 | 批量实验 | 设置轮数并点击开始 | 多轮实验依次运行并输出汇总文件 | 待测 |
| F-14 | 自定义导出目录 | 选择或输入目录并应用 | 导出根目录切换成功 | 待测 |
| F-15 | 相机切换 | 总览/跟随/下一架无人机切换 | 镜头状态与目标显示正确 | 待测 |
